# Sommnus

Sommnus는 사용자의 수면 효율을 극대화하기 위해 90분 단위의 렘(REM) 수면 주기를 기반으로 최적의 기상 및 취침 시간을 계산해주는 웹 애플리케이션입니다.

## 📚 가이드 문서
[백엔드 가이드 문서 보기](https://github.com/Bona-Stella/backend-docs-hub/blob/main/somnus/README.md)

## ✨ 주요 기능
- **수면 시간 계산**: 기상 희망 시간을 설정하면 최적의 취침 시간을 추천합니다.
- **기상 시간 계산**: 취침 희망 시간을 설정하면 개운하게 일어날 수 있는 기상 시간을 추천합니다.
- **개인화 설정**: 잠드는 데 걸리는 시간과 수면 주기 단위를 사용자 맞춤형으로 설정할 수 있습니다.
- **PWA 지원**: 모바일 환경에 최적화된 설치형 웹 앱(Progressive Web App) 경험을 제공합니다.

## 🛠 기술 스택
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **State Management**: React Hooks
- **PWA**: @ducanh2912/next-pwa

## 🚀 시작하기 (Getting Started)

프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따르세요.

### 설치

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 실행

개발 서버를 실행합니다:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.
